## Select, Optgroup, Option, Datalist 태그에 관한 내용

#### `<select>`
옵션을 선택하는 메뉴

 속성 | 의미 
---|:---:|---:
`autocomplete` | 사용자가 이전에 입력한 값으로 자동 완성 기능을 사용할 것인지 여부
`disabled` | 선택 메뉴를 비활성화
`form` | `<form>` 후손으로 `<select>`가 없을 때 `<form>`과 연결하고자 할 때 사용한다,
`multiple` | 다중 선택 여부
`name` | 선택 메뉴의 이름
`size` | 한 번에 볼 수 있는 행의 개수
